# TO_DO_LIST
# To-Do List Application

A simple web-based To-Do List application built using HTML, CSS, and JavaScript.

## Features
- Add new tasks dynamically
- Edit existing tasks
- Mark tasks as completed
- Display task status (Pending / Completed)

## Technologies Used
- HTML
- CSS
- JavaScript

## Description
This project helped me understand DOM manipulation, event handling, and basic frontend logic using JavaScript
